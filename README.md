隐私政策

最后更新：2023年12月8日

感谢您使用 Network Status Monitor。本隐私政策旨在说明我们如何处理您的信息。

信息收集和使用
• 本应用不收集任何个人信息
• 本应用不需要网络连接
• 所有功能均在您的设备本地运行

权限使用说明
本应用仅使用以下必要权限：
• ACCESS_NETWORK_STATE：用于监控网络状态变化
• ACCESS_WIFI_STATE：用于获取 Wi-Fi 信息
• POST_NOTIFICATIONS：用于发送网络状态变化通知

数据存储
• 本应用不存储任何用户数据
• 所有操作仅在内存中临时进行

第三方服务
• 本应用不使用任何第三方服务
• 不与任何第三方共享数据

变更
如本隐私政策有任何变更，我们将在此页面更新内容。
